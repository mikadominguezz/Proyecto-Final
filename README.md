
# Marketplace de Servicios

This is a code bundle for Marketplace de Servicios. The original project is available at https://www.figma.com/design/IZdNsNe8N0Zj64EfZaZ3bT/Marketplace-de-Servicios.

## Running the code

Run `npm install` to install the dependencies.

Run `npm run build`

Run `npm run dev` to start the development server.
